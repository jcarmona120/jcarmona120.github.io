### THE TERMINAL

This sounds silly but I love the terminal. There are a ton of commands and you have so much to learn but it's so satisfying. I enjoy the ease of the Terminal. I enjoy the power of that little box. It's an essential tool to get you to the next level as a developer. The power it gives you is incredible. I was excited early on just making folders with "mkdir". Now I've been able to dive into Node and launch a server to view my local environment come to life. That is extremely exciting. "webpack --progress --colors --watch" is so cool. Automatic updating and bundling of your css and javascript files! Here are my top 5 commands right now:

    1) git commit -m "A Wonderful Message"
    2) open -a "Chrome" index.html
    3) nodemon app.js
    4) npm install
    5) npm start

That's not in any order. Just some of the fun commands I'm starting to use more and more.
