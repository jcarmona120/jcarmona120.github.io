### What I Learned Today

**React** -
I've been dealing with several different tutorials over the past few months. Most important thing I learned was that React.createClass was deprecated and adds a console warning in your browser. 

**Node** -
I've been trying to get used to reading the documentation for Node and really diving into Express middleware. It's endless and I really enjoy the possibilities. 


**Pug** _and_ **markdown** -
I love Pug because it's markdown meets Javascript. That's awesome. And I didn't really get into Markdown until Pug. 

**Stateless API** -
Quick explanation of Stateless in REST API's: No memory of any previous requests. I'm sure I could make that more eloquent but the point is to make things simple. That's simple.

Yeah, I paid for this course because I really enjoyed his Javascript30 course(array cardio!). It's pretty exciting how he pushes you along (sometimes I feel like it's too fast but he winds up showing you how things work). 

[Frontendmasters](www.frontendmasters.com) -
This is my go-to source right now for in-depth workshops. I'll write up something on this soon. I'd recommend this subscription more and more just to get the experience of listening to experienced developers explain how things work.

I love this website as well. When I'm not getting things from documentation or more advanced tutorials, this is my foundation. I'd say it gave me a great foundation for Front End Development as well as a great resource learning Wordpress.

I've done the free course and the premium content. The best thing I've gotten from this course is how to read Javascript and use the console.
