### Overload

I try to cram so much stuff into my brain. Learn so much while I'm at work. Videos, articles, looking over notes. Revisiting older projects. It can feel overwhelming sometimes yet it's still very exciting. I've been approaching teaching myself coding as if I were in school. I've tried to avoid spending more than three hours on one subject. For instance, I spent the morning watching React videos. The afternoon was spent on Wordpress. This really reflects the different projects I'm working on. I finished the React tutorial on Treehouse when I got home. Finishing the night with some Jekyll work and this blog. So this blog is what it's like in my brain now. All over the place. I'll give this some more thought tomorrow. I'll also start working on my first full tutorial for this site.

### What I Learned Today

**React**

The Treehouse tutorial actually did a great job of making sure to include array functions like map and reduce. It's nice to see those things in action in the middle of a React project. That's not so much about React but its nice to continue to see more fundamentals like that in the middle of the framework. Can't stop studying the basics and how to apply them.

**Wordpress**

The Query Monitor Plugin seems like a super useful plugin. Didn't get to play with it but its one of those tools that gives you the chance to really pull everything apart and understand it deep down. Excited to check out developed themes and break them down to learn more.

**Jekyll**

As I continue to build my blog on Jekyll and work with frameworks like React, you learn more and more about this idea of breaking up code into partials. Over the next week, I'll be breaking up the site more and more and making the code more readable.
